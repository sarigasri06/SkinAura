# SkinAura

A full-stack skincare e-commerce website built using the MERN Stack.

## Features
- User Login & Registration
- Product Categories
- Product Search
- Shopping Cart
- Checkout
- Order Tracking
- Admin Dashboard
- Before & After Gallery
- Skin Journal
- Rewards & Loyalty
- Multi-language Support
- Notifications

## Tech Stack
- Frontend: react.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Styling: CSS
